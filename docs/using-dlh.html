Listing information about SOMA Data Source Name

In pd.read_parquet() bellow the path object is a string and could be a URL. Valid URL schemes include http, ftp, s3, gs, and file. For file URLs, a host is expected. A local file could be: file://localhost/path/to/table.parquet. A file URL can also be a path to a directory that contains multiple partitioned parquet files. The pyarrow API support paths to directories as well as file URLs. A directory path could be: file://localhost/path/to/tables or s3://bucket/partition_dir.

filelist = [
    'soma-10m-0088833.parquet',
    'soma-10m-0088834.parquet',
    'soma-10m-0088835.parquet',
    'soma-10m-0088836.parquet'
]
print(len(filelist), 'Arquivos na lista')

import pandas as pd
pd.options.display.float_format = '{:.9f}'.format

def list_dsuuid(f):
    df = pd.read_parquet(f, engine='pyarrow', columns=['dsuuid', 'ts', 'value'])
    print(df['dsuuid'].describe())
    return len(df['ts'])

# - - - - - - - - - - - - - - - - - - - - - - - - - - - -
qty = 0;
for f in sorted(filelist):
    if f.endswith(".parquet"):
        qty = qty + list_dsuuid('../data/reusable/' + f)

print("qty =", qty)
print("FIM")
