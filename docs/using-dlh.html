Using SOMA Data Lakehouse

To use SOMA Data Lakehouse you need some experience in at least one of the following programming languages:

  • Java
  • Python
  • Rust
  • SQL

Below is an illustrative example of how to use SQL with Arrow and Datafusion in Python language.

#!/usr/bin/env python3

import jq
import pandas as pd
import pyarrow as pa
from datafusion import SessionContext
ctx = SessionContext()
# There is a thousand parquet files in this directory
d = 'data/reusable/'
ctx.register_parquet("soma", d)
sql_cmd = """
 SELECT date, ts, dsuuid, value
   FROM soma
  WHERE ts > 1663368556001001
"""
rb = ctx.sql(sql_cmd).collect()
t1 = pa.Table.from_batches(rb)
df = t1.to_pandas()
first_five = df.head()
print(df.describe())
print(first_five)
t2 = pa.Table.from_pandas(first_five)
print(jq.compile('.').input(t2.to_pydict()).text())
